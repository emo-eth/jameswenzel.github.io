Personal Projects

InstaCaption

Retrieves song lyrics from an artist based on a supplied topic. Suggests similar topics and lyrics using a semantic space. Built at the Science of Music Hack Day, hosted by ISMIR and Spotify in NYC.

Python - Beautiful Soup, NLTK, Semantic Spaces, Word Vectors

Drake Hate Twitter Bot

Heroku app that uses the Twitter API to retweet mean tweets about the recording artist Drake using a naive-Bayes classifier trained on a custom dataset using NLTK in Python.

Check us out @sixgodwatching.

Color Synth Filter

Maps a user-picked color to the frequency and intensity of a subtractive audio filter to an in-browser synthesizer. Built as part of the Multisensory Music Hack Day at Spotify in NYC.

HTML, CSS, Javascript - tone.js

Color-Mapped Audio-Reactive LED Screen

Hand-built 16x32 LED screen, driven by a microcontroller, linked to Processing via serial bus. Reacts to audio analyzed via Fast Fourier Transform and maps color to pitch based on peak perceptual sensitivity.

Processing/Java - FFT, Arduino
